# cordova-plugin-jshare
JPush 官方支持的社交分享 Cordova 插件，具备国内主流社交平台分享功能，提供接入新浪微博，QQ，微信等第三方社会化分享服务，同时支持 Android 与 iOS 平台。

功能特性：
- 集成简单。
- 稳定，安装包小，不依赖任何第三方库。
- 社会化统计分析。完整的统计和分析后台，帮助开发者了解多种统计指标。
- 一键分享。支持对新浪微博、QQ、微信等社会化平台一键分享；分享内容包括文字、图片、链接、音视频、文件、表情等。
